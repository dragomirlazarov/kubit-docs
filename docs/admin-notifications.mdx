The Admin panel is the primary area where users with Admin level permissions can manage:

* Notifications of certain events in your Kubit environment
* User Access to your Kubit environment.
* User Roles to define various types of access users can have.
* User Groups to easily assign users to the correct group of Roles and Permissions.
* You can find the Admin Panel in the top left under user options. If you are not an Admin you will not see this option.

# Notification Panel

When you enter the Admin Notification panel you will see:

* A place to enter the email address for Email notifications to be sent to.
* Option to integrate with Slack for Slack notifications (must be enabled by your Slack admin).
* A list of events for which you can be notified.
* The type of notification: Email, Slack or In-Product.

# Configuring Notifications

## Adding an email address

Start by typing your email address and then pressing the enter key. The newly added email address will appear under the box where you just typed it.

![](/images/docs/c84d4b7a9a3c4405227db8bac1f606c78047cd48bc395e559beba1e6d6ce78f2-adding_an_email_address.gif)

## Configuring Slack

Click on Connect to Slack... then choose to "Allow" when prompted for permissions. Select the desired channel.

![](/images/docs/7c8cb0bc7b6e631329342d9ce6f84a2510327e6e2e1179406d44ec4b5bd4d422-configuring_slack.gif)

Once configured you'll receive this message in Slack.

![](/images/docs/7b538fd9b94313d4b0565216d0b454b67414b1e41be3e161cd6ae5c7762dc673-image.png)

## Select Events

Determine which event to be notified for, and select whether to be notified via email, Slack (if configured) or in-Product. When ready, select Save at the bottom.

![](/images/docs/5d5bdb73d9a127808cc58caf368742a1acebd154d818b41335eddf08c878f456-select_events.gif)

# Viewing Notifications

## In-Product Notifications

New Notifications will appear in-Product, denoted by a red dot on the bell. Click on the blue dot (beside the notification) to mark it as read. Clicking on the item will open the item in question.

![](/images/docs/5ecb84906f49a4f51c3cd5a1c48e847895ed6491ae2fabc16701c56434491f48-in-product_notifications.png)
