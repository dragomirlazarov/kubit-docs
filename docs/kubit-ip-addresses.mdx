## IP Whitelist

Enhance the security of your data by controlling access through IP whitelisting.

### IP Ranges

Kubit’s IP ranges are available at [https://ip-ranges.kubit.ai](https://ip-ranges.kubit.ai). Regularly update your firewall rules to accommodate any changes.

### Retrieve IP Ranges with cURL

To fetch the latest IP ranges, use the following cURL command:

<CodeGroup>
  ```bash bash
  curl -s https://ip-ranges.kubit.ai
  ```
</CodeGroup>
