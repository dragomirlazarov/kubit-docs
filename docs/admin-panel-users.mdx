# Users Page

To manage Users first find the `Settings` page in left navigation bar by clicking on the ⚙️ icon and then select `Users`.

![](/images/docs/fd1cfa7a73e8a6a9d0f3b3a479ad4a63ae4c8592452c9850f4e09085df30e2c3-users_page.png)

Here you will see a full list of users who have logged into Kubit. It will also show the `Last Login` and `Joined On` dates to help you better understand which users are new and which are active.

## Adding Users

Access to Kubit is typically provisioned through SSO. Once the user has been granted access through your organization they will be able to log into Kubit and will appear on this list after first login. However, you can also invite users to pre-provision their default schema, view and role by clicking on the `Invite New User` blue button near the top right of the Users page.

![](/images/docs/60680c3ef3e56b8367583984161aba8e5825e940018df052c1b7e96b9d0feaf4-Screenshot_2025-10-23_at_23.32.45.png)

## Viewing Users

Click on the vertical 3 dots to the far right side of a user and select "View User". This will give you more detailed information about that user and the [Groups](doc:admin-panel-groups) and [Roles](doc:admin-panel-roles) they have assigned to them.

![](/images/docs/70197bf61bbf7abb2b1009f30a0ec259f65bfda75a33bd5672d969e59ca5db55-Viewing_users_2.png) ![](/images/docs/ad022bfb6f900257c37f7d97e97fbcef5b2f2d91a1f9d465ace79c0605218d0e-Screenshot_2025-10-23_at_23.37.48.png)

## Edit User

Once you've selected `View User` you are able to change that user's permissions by using the edit pencil in the upper right corner.

You are able to assign `Groups` and `Roles` to the user.

![](/images/docs/a348405c1c92849e3a8a6e35dc9b36c6ba65d8005cdd8057ff554dc295555c50-edit_user.gif)

## Disable/Re-enable Users

From the main Users page and the same vertical 3 dots menu you are also able to Disable a user from being able to log into Kubit. Once you do that you will be asked to transfer all existing entities such as Reports and Dashboard to another user who will be able to manage them.

If you choose to re-enable them follow the same flow and select `Enable User`.

![](/images/docs/afbe3ebeb8e629808a50d3b6c0c2f34302a4eb9fd397c2f14faf26f2326ffdb2-disable_user.png)
